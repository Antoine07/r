# Mini-projet — Bases R 

Ce mini-projet sert à pratiquer les bases de R pour découvrir R, en suivant une structure de projet simple.


## Structure

- `data/raw/`: données sources
- `r/`: scripts
- `output/`: sorties générées (tables + figures) - Nous verrons ça plus tard.

## Démarrage

1. Ouvrir `mini_projet_bases_r.Rproj`. (double cliquer)
2. Exécuter le script de travail:

```r
source("r/01_setup.R")
```

Un corrigé est disponible:

```r
source("r/sandbox.R")
```
